# Chrome Extension (MV3)

## Como carregar no Chrome (DEV)
1. Chrome → `chrome://extensions`
2. Ative `Modo do desenvolvedor`
3. Clique `Carregar sem compactação`
4. Selecione a pasta `chrome-extension`

## Como usar
1. Clique ⚙️ e configure `Cloud Run API URL` (ex.: https://SEU-SERVICE.a.run.app)
2. (Opcional) configure `Client Key` se você ativar `CLIENT_KEY` no servidor.
3. Volte e escolha um plano → `Abrir portal` → faça login manualmente
4. Copie o script → F12 → Console → Cole e execute

> Esta extensão NÃO coleta login/senha. Você controla tudo manualmente no portal.
